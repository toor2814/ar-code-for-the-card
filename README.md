# Sussy Imposter — WebAR

An augmented reality web application built with Three.js and WebXR. Place a 3D model of the "Sussy Imposter" character into your real-world environment using your device camera.

## How It Works

1. Open the page on a WebXR-compatible device (Android Chrome, iOS Safari 16+, or desktop Chrome)
2. Tap the AR button to activate the camera
3. Tap on a surface to place the 3D model
4. Move around to view the model from any angle

## Tech

- Three.js — 3D rendering engine
- WebXR — AR hit-test API for surface detection
- GLTFLoader — 3D model loading (.glb format)

## Files

- `index.html` — Main AR application
- `SussyImposter.glb` — 3D model
